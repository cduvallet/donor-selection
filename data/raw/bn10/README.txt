This local folder contains metabolomics data from healthy stool 
donors included in the Broad Next 10. The excel file is too large
to be pushed to github and so is only available locally.

Claire received these raw data from Sean Gibbons on Jun 19, 2018,
via personal communication.

*These data are not public and should not be used without explicit*
*permission from the original authors (i.e. Sean Gibbons et al)*
