### Downloading Data

Metadata was downloaded from the ENA.

Importing bulk download file from EMBL and converting to a single dimensional list of ftp strings

```
import pandas as pd

longman2017_prj = pd.read_csv('Longman2017_PRJNA388210.txt', sep='\t')
longman2017_allgz = longman2017_prj['fastq_ftp'].str.cat(sep=';')
with open('longman2017.fastq_ftp.txt', 'w') as f:
     f.write('\n'.join(longman2017_allgz.split(';')) + '\n')
```

Now, download:

```
wget -P data/ -i longman2017.fastq_ftp.txt
```
