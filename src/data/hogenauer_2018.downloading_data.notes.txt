### Downloading Data

Metadata file was downloaded from the ENA, with some additional columns selected (e.g. fastq ftp)

Importing bulk download file from EMBL and converting to a single dimensional list of ftp strings

```
import pandas as pd
hogenauer2018_prj = pd.read_csv('Hogenauer2018_PRJEB11841.txt', sep='\t')
hogenauer2018_allgz = hogenauer2018_prj['fastq_ftp'].str.cat(sep=';')
with open('hogenauer2018.fastq_ftp.txt', 'w') as f:
     f.write('\n'.join(hogenauer2018_allgz.split(';')) + '\n')
```

Now, download:

```
wget -P data/ -i hogenauer2018.fastq_ftp.txt
```
